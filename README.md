## 🧱 Field summary

- `domain` — Fully qualified domain name.
- `status` — Current acquisition state for the domain in the public extract.
- `purchase_price` — Visible purchase price when available.
- `renewal_price` — Visible renewal price when available.
- `attractiveness` — Composite naming score used as a decision-support signal.
- `demand` — Relative buyer-pressure score when available.
- `length` — Character count without the TLD.
- `registrar` — Registrar name when known.
- `created_at` — Creation timestamp when known.
- `expires_at` — Expiry timestamp when known.

## ⚠️ Methodology and caveats

This set is defined by automotive keyword matching, so it spans a wide range of exact terms and adjacent words rather than a single naming style. The sample includes direct vehicle terms such as car, suv, and bicycle, along with broader words like utility, technology, and door in varied extensions. With 22,056 domains across 505 TLDs, the main decision is not just the word but how well the extension supports commercial intent, memorability, and renewal discipline. Median ask is $1,206, which gives a useful pricing midpoint, but individual names can differ sharply based on term strength and TLD credibility.
